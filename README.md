# NoCSS Framework

> A classless CSS framework that styles semantic HTML intelligently using CSS selectors, `:has()`, and contextual detection. Zero classes needed.

[Live Demo](https://dathims.github.io/nocss) ‚Ä¢ [Documentation](docs/README.md) ‚Ä¢ [Components](https://dathims.github.io/nocss/components.html)

## Quick Start

### CDN

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@dathims/nocss/dist/nocss.min.css">
```

### npm

```bash
npm install @dathims/nocss
```

```html
<article>
  <h2>Card Title</h2>
  <p>Automatically styled as a beautiful card.</p>
  <footer>
    <button>Action</button>
  </footer>
</article>
```
