# NoCSS Framework

> A classless CSS framework that styles semantic HTML intelligently using CSS selectors, `:has()`, and contextual detection. Zero classes needed.

## Quick Start

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/nocss-framework/dist/nocss.min.css">
```

```html
<article>
  <h2>Card Title</h2>
  <p>Automatically styled as a beautiful card.</p>
  <footer>
    <button>Action</button>
  </footer>
</article>
```
