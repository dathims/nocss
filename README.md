# NoCSS Framework

> A classless CSS framework that styles semantic HTML intelligently using CSS selectors, `:has()`, and contextual detection. Zero classes needed.

[Live Demo](https://dathims.github.io/nocss) ‚Ä¢ [Documentation](docs/README.md) ‚Ä¢ [Components](https://dathims.github.io/nocss/components.html)

## Quick Start

### CDN (unpkg)

```html
<!-- CSS -->
<link rel="stylesheet" href="https://unpkg.com/nocss-framework@latest/dist/nocss.min.css">

<!-- Optional: JavaScript helpers -->
<script src="https://unpkg.com/nocss-framework@latest/dist/nocss.js" defer></script>
```

### npm

```bash
npm install nocss-framework
```

```javascript
// Import in your JavaScript
import 'nocss-framework/dist/nocss.css';
```

### Download

Download the [latest release](https://github.com/dathims/nocss/releases/latest) and include the CSS file:

```html
<link rel="stylesheet" href="path/to/nocss.min.css">
```

```html
<article>
  <h2>Card Title</h2>
  <p>Automatically styled as a beautiful card.</p>
  <footer>
    <button>Action</button>
  </footer>
</article>
```
